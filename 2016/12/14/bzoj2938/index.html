「BZOJ2938」「Poi2000」病毒

发表于 2016-12-14 | 分类于 题目精选

相关链接

题目传送门

解题报告

题解：对于一组病毒编码，我们构建AC自动机。如果存在一个无限长的安全代码，它在里面匹配，将一直匹配不上，也就是说，失配边会形成一个环。
所以构建好AC自动机后dfs失配边看有无环即可。

Code

# AC自动机 # DFS
